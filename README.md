# Image Processing Lab -- OpenCV (Python)

## Project Overview

This Jupyter Notebook demonstrates basic image processing techniques
using OpenCV, NumPy, and Matplotlib in Python. The notebook allows users
to upload a grayscale image and apply different image transformation
operations. It is designed for learning fundamental concepts of digital
image processing.

## Features

-   Upload an image directly in the notebook
-   Convert and process grayscale images
-   Apply image transformation techniques
-   Visualize results using plots

## Technologies Used

-   Python
-   OpenCV (cv2) -- for image processing
-   NumPy -- for numerical operations
-   Matplotlib -- for displaying images
-   Google Colab file upload utility

## Steps Performed in the Notebook

1.  Import required libraries.
2.  Upload an image using the file upload tool.
3.  Read the image in grayscale format.
4.  Apply image processing operations such as:
    -   Negative image transformation
5.  Display the original and processed images using Matplotlib.

## How to Run

1.  Open the notebook in Jupyter Notebook or Google Colab.
2.  Run the first cell to upload an image.
3.  Execute the remaining cells sequentially.
4.  View the processed results in the output plots.

## Example Output

The notebook shows a comparison between: - Original grayscale image -
Processed image after transformation
